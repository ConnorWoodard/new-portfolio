Sports Pro

Project Information

This project had me going back to C# while using JS and HTML. This project is the SportsPro site. It is a site that allows for users of various software to report issues with the software. The site allows admins to check the incidents as well as close them and add new customers.

Problem Solved

The problems with this project were having to use authentication/authorization as well as publishing the site out to Azure. The authentication/authorization was a problem because I had to make sure that the correct users could do the correct actions. The publishing was a problem because I had never published a site to Azure. I encountered problems with the connection string and the database. I was able to overcome these problems by dropping the database and recreating it. I also had to make sure that the connection string was correct. I was also able to learn about ViewModels as well as various configuration settings in the appsettings.json file.
