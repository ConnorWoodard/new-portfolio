Games of the Year

Project Information

This project was using HTML, CSS, JS and ReactJS to create an application that would follow CRUD operations in a database. CRUD being Create, Read, Update, Delete. My project was making an application that people could add/delete games they think deserve to be game of the year for various years.

Problem Solved

The problems encountered on this project were not only learning ReactJS so close to the deadline but also making the entire project work on only one page. All of our other projects in class were multi page applications. This one needed to Create, Read, Update, and Delete all on one page. This was a challenge but I was able to overcome it.
