Issue Tracker

Project Information

This project had me use many different languages and dependencies. Some of these include; JS, Bootstrap, CSS, HTML, ReactJS, NodeJS, ExpressJS, MongoDB, Google Cloud, and NodeMailer. This project was titled Issue Tracker. It was a project that would allow users to register, login, and report bugs on a website. Users with the correct permissions could delete, test, and close bugs.

Problem Solved

The problems on this project were using roles for different actions, using a dependency that we have not covered, and deploying the project to Google Cloud. The roles were a problem because I had to make sure that the correct users could do the correct actions. The dependency was a problem because I had to learn how to use it on my own. The deployment was a problem because I had never deployed a project to Google Cloud before. I was able to overcome these problems by using the documentation provided by the dependencies and Google Cloud. I was also able to find a very interesting dependency called NodeMailer that allowed me to send users codes in an email to reset passwords.
